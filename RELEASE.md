# RELEASE

## Version 0.2.0

### First public candidate
This release establishes the public `molt_evo` runtime surface.

### Included
- `molt_evo_*` primary runtime scripts
- `molt-evo-*` service and timer units
- unified runtime data root at `.openclaw/molt_evo`
- persistent local memory database
- structured daily summaries
- conversation ingestion
- normalization and conflict detection
- conflict alert / notify / feedback workflows
- backup / export / import support
- install / upgrade / uninstall / bootstrap / healthcheck / selftest

### Release note
This repository is suitable as a public candidate release for OpenClaw-oriented local runtime memory workflows. Historical migration material is intentionally kept outside the strict public release surface.
